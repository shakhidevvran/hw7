***Что такое структуры данных?***
Структура данных - это структуры, которые хранят данные вместе. в Питоне есть четыре встроенных
структуры данных - кортеж, словарь, список, сеты.


***Зачем нужны структуры данных?***
для хранения связанных данных :/



***Что такое Hash Map?***
Hash map - это тип структуры данных, которая хранит данные в паре "ключ-значение".


***Что такое Linked List?***
Linked List - это связной список,то есть стурктура данных, состоящая из элементов, в которой хранятся данные,
которые в свою очередь соеденены между собой связями.


***В каких случаях использовать Linked List и Hash Map?***
Linked List стоит использовать, когда происходит вставка или удаление элемента с середины списка.
Hash map - используется почти также, то есть для получения данных мы используем определенный ключ. Так же, для быстрого и
легкого извлечения элемента по его ключу.
